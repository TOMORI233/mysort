------------------------------------------------------------------------------------------------------
-------------------------------------- ABOUT THE TOOLBOX ---------------------------------------------
------------------------------------------------------------------------------------------------------
- Before you use this small toolbox I strongly encourage you to read the original paper on the       -
- gap statistic algorithm by Tibshirani, Walther & Hastie (2001). Also, you can find a great         -
- summary on Edwin Chen blog, here: http://blog.echen.me/2011/03/19/counting-clusters/               -                                                                         -
-                                                                                                    -
- The only file you need to run is gap_statistics; that function will call the other two functions.  -
-                                                                                                    -
- All the files are commented, so you shouldn't have any problems understanding how they work. The   -
- function was written to be run in parallel, in case you have the Parallel Computing Toolbox. If    -
- you don't have it, that's not a problem: just leave the fourth argument blank.                     -
-                                                                                                    -
- I have also included some fabricated data, in case you want to test the toolbox.                   -
-                                                                                                    -
- Just try: [opt_k, gaps] = gap_statistic(test_me, 1:10, 10);                                       -
-                                                                                                    -
- (By the way the correct answer is 4.)                                                              -
-                                                                                                    -
- Have fun!                                                                                          -
